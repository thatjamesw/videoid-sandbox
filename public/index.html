04

          
Signicat mapping

          
Quick reference for the three requests this tool orchestrates.

        


        

          

            

              
Document support

              
`GET /assure/{provider}/document-types` fetches the numeric IDs you can allow.

            

            

              
Configuration

              
`PUT /assure/capture/configurations/{configurationId}` updates the capture configuration.

            

            

              
Capture launch

              
`POST /assure/dossiers/{dossierId}/capture` starts the flow and can include `captureParameters.uiProfile`.

            

          

          

            On GitHub Pages, credentials are stored only in this browser session and calls are made
            directly to Signicat. If a direct request fails before reaching Signicat, check that CORS
            allows this page origin.
